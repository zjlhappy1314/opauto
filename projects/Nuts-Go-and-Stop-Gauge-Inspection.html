Overview

The Openex Nuts Go-and-Stop Gauge Inspection Machine is a standard, high-speed automatic inspection system designed for M4-M16 nuts.
It replaces manual thread gauging with a dual-gauge rotary torque detection mechanism, dramatically improving efficiency and accuracy.

Operators simply load parts into the vibrating feeder bowl — from there, the system automatically feeds, inspects, classifies, and sorts the nuts without manual intervention.
With a touchscreen HMI interface, operators can easily configure torque parameters, view inspection data, count output, and monitor alarms.

This machine is particularly suited for:

  • Automotive fastener nuts
  • Weld nuts
  • Copper and titanium electronic nuts
  • Precision threaded components for general manufacturing

Key Features

Dual-Gauge Rotary Mechanism

Dual go-and-stop gauges feed nuts simultaneously into the rotary disk inspection station, ensuring high positional accuracy and stable operation.

High-Precision Torque Measurement

Reads and controls the torque of go/no-go insertion with 0.01N·m accuracy, providing reliable torque data for every piece.

Fully Automated Workflow

Automatic feeding, detection, and sorting reduce manpower requirements. One operator can manage multiple machines simultaneously.

Smart User Interface

Intuitive full-touch HMI panel guides operators through setup and real-time monitoring. Built-in counting and alarm systems simplify production management.

OPENEX AI Vision Integration

Optionally integrates with the Openex AI Vision system for real-time image analysis and defect detection on the nut surface.

Flexible, Product-Safe Design

The flexible threading mechanism guarantees alignment and prevents product damage during torque testing.
