Overview: What's it?

The Openex Automotive Parts Inspection Machine is a non-contact, high-speed visual inspection system built specifically for the automotive manufacturing industry. It uses industrial cameras, lighting systems, and deep learning software to detect a wide variety of defects in metal, plastic, and composite parts ‚Äî even at micro-level resolution.
It supports inline or offline integration and can be customized for small precision components or large structural elements. With deep learning algorithms, the system can accurately identify even unpredictable or rare defect types ‚Äî minimizing false positives and negatives.

Key Features & Capabilities

  • AI-Based Defect Detection
    • - Leverages deep neural networks for adaptive learning
    • - Easily handles complex geometries, irregular surfaces, and varying materials
    • - Detects subtle defects like micro cracks, burrs, deformation, and foreign particles
  • Comprehensive Inspection Coverage
    • - Surface Inspection: Scratches, dents, inclusions, blisters
    • - Dimensional Checks: Length, width, height, depth, hole size, spacing
    • - Form Accuracy: Warping, ovality, flatness, thread completeness
    • - Assembly Verification: Correct positioning, component presence, orientation
    • - Color and Code Recognition: OCR, barcodes, markings, labels
  • Flexible Mechanical Integration
    • - Supports rotary tables, linear conveyors, robotic part handling, vibratory feeders
    • - Multi-angle camera configurations for complete inspection coverage
    • - Can be retrofitted into existing production lines or delivered as turnkey station
  • Data Logging & Traceability
    • - Real-time results display with NG/OK classification
    • - Stores images, defect reports, inspection history for traceability
    • - Ethernet/PLC/OPC UA communication for MES integration
  • Auto-Sorting and Rejection System
    • - High-speed pneumatic or robotic rejection for failed parts
    • - Multiple NG bins configurable by defect type

Typical Automotive Components Inspected

Openex's inspection system is compatible with a broad range of automotive parts:

  • Plastic Fasteners: Clips, retainers, push rivets
  • Metal Hardware: Bolts, washers, spring pins
  • Connectors & Terminals: Electrical plug housings, metal contacts
  • Bushings & Grommets: Rubber bushings, composite inserts
  • Precision Castings: Exhaust parts, heat shields
  • Interior Components: Switch housings, knobs, bezels
